# Assets requeridos — colocar en esta carpeta

## Logo (REQUERIDO)
- **Archivo**: `fauno_logo.png`
- **Fuente**: El logo teal "fauno" sobre fondo negro que compartiste
- **Requisito**: PNG con fondo transparente (si es posible) o fondo negro
- La imagen que compartiste en el chat es perfecta tal cual

## Fuentes (REQUERIDAS)
Descargar desde Google Fonts:
- `Nunito-ExtraBold.ttf` → https://fonts.google.com/specimen/Nunito
- `DMSans-Regular.ttf` → https://fonts.google.com/specimen/DM+Sans

## Icono de check (opcional)
- `icon_check.png` — ícono pequeño para bullets de beneficios (20x20px)
- Si no se agrega, el script usa "✓" como texto

## Estructura final esperada:
```
config/
└── assets/
    ├── fauno_logo.png        ← el logo que compartiste
    ├── icon_check.png        ← opcional
    └── fonts/
        ├── Nunito-ExtraBold.ttf
        └── DMSans-Regular.ttf
```
